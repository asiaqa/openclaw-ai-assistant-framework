### 2. 九宫格专项评估标准

#### 2.1 一致性评分 (1-5分)

| 评分 | 描述 |
|------|------|
| 5 | 9张图角色/风格完全一致，只有目标变体不同 |
| 4 | 基本一致，轻微差异不影响识别 |
| 3 | 中等一致性，有明显差异但仍可辨认 |
| 2 | 差异较大，勉强能看出是同一主题 |
| 1 | 完全不一致，像不同图片拼凑 |

#### 2.2 变体质量评分 (1-5分)

评估每个子图独立质量：
- 手部结构准确性
- 面部细节
- 光影一致性
- 构图平衡

### 3. 九宫格Prompt优化测试

```python
# 写法A: 直接要求
"generate 9 images in a 3x3 grid showing..."

# 写法B: character sheet术语

# 写法C: 分步描述
"9 images: 1) front view 2) side view 3) back view..."

# 写法D: 强调一致性
"consistent character across 9 images..."
```
